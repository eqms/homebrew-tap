# Equitania Homebrew Tap

Homebrew formulas for Equitania Software GmbH tools.

## Install

```bash
brew tap eqms/tap
brew install actop
```

or in one step:

```bash
brew install eqms/tap/actop
```

## Formulas

| Formula | Description |
|---|---|
| [actop](https://github.com/eqms/actop) | Top-like terminal UI for Apple containers (like ctop) |
